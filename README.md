# Bingo Game - Многопользовательское приложение

Веб-приложение для игры в Бинго (1-90) с несколькими участниками.

## Технологии

- **Frontend**: HTML, CSS, Vanilla JavaScript
- **Backend**: Node.js, Express, Socket.io (WebSocket)
- **Storage**: In-memory (можно заменить на Redis)

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Или для development:

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

## Использование

1. **Создание игры (Host)**:
   - Откройте http://localhost:3000/host
   - На экране появится QR-код для подключения участников
   - Нажмите "Начать игру" когда все участники готовы

2. **Подключение участника (Player)**:
   - Отсканируйте QR-код с экрана Host
   - Заполните карточку 5×5 числами от 1 до 90 (без повторений)
   - Нажмите "Сохранить карточку"
   - После начала игры числа будут автоматически зачеркиваться

3. **Управление игрой (Host)**:
   - Нажимайте "Следующее число" для выпадения нового числа
   - Все участники получат обновление в реальном времени
   - Используйте "Начать заново" для сброса игры

## API

### REST Endpoints

- `POST /api/session/create` - Создать новую игровую сессию
- `GET /api/session/:id` - Получить информацию о сессии
- `POST /api/session/:id/card` - Сохранить карточку игрока

### WebSocket Events

**От клиента:**
- `join_session` - Присоединиться к сессии
- `start_game` - Начать игру
- `draw_number` - Вызвать следующее число
- `reset_game` - Сбросить игру

**От сервера:**
- `session_started` - Игра началась
- `number_drawn` - Выпало новое число
- `session_reset` - Игра сброшена
